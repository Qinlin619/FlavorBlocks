# PIXEL SLICE (环球美食版)

一款极简、解压的像素不规则拼图游戏。

## 游戏简介
**PIXEL SLICE** 是一个以“环球美食”为故事线的拼图游戏。玩家通过将散落的像素块解构并重新拼凑，在黑白与色彩的交织中探索世界各地的特色美食。

## 核心特性
- **美食叙事**：从简单的亚洲饭团到复杂的西洋汉堡，关卡难度循序渐进。
- **不规则切片算法**：每一局拼图都会由算法动态生成不规则形状，提供独特的挑战感。
- **色彩动力学**：拼图过程中，食材的色彩会随着拼凑进度逐渐亮起。
- **自动吸附**：保留了极致解压的吸附反馈感。
